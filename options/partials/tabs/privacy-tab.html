    Privacy Policy

    Last updated: April 2026
    Read Full Policy

    Overview

    TimeDash runs locally in your browser. It does not include analytics, telemetry, ads, or tracking scripts. There is no login and no server that receives your browsing history.

    Data Storage

    TimeDash stores data in your browser profile using Chrome's chrome.storage.local API. This storage is scoped to the extension and your profile. It can include:

    • Time tracking records and usage history
    • Playback speed preferences
    • Site blocking and restriction rules
    • Theme and UI customization settings

    TimeDash reads this data to show your stats and apply your rules. It does not send it to remote services.

    Network Activity

    TimeDash does not make API calls to a server. If you see favicon requests in DevTools, those are created by the browser's image handling. In Chrome, they may be routed through Google's favicon service. That traffic is not generated by TimeDash code.

    Permissions & Justification

    • storage lets TimeDash save settings and usage records.
    • tabs lets TimeDash read the active tab URL so it can attribute time to the right site and apply block or restriction rules.
    • scripting lets TimeDash inject video speed controls into pages that use HTML5 video players.
    • webNavigation lets TimeDash notice navigations in frames so tracking and blocking rules stay accurate after redirects and iframe loads.
    • host_permissions (<all_urls>) lets the content scripts run on the sites you visit. Without it, tracking and video controls would only work on a limited set of domains.
    • alarms lets TimeDash run scheduled maintenance like daily resets and optional auto purge.
    • notifications lets TimeDash show optional alerts for time limits and tracking pause state.

    Data Deletion

    You control your data. You can delete it at any time from within the extension, and uninstalling the extension removes the stored data from this browser profile.

    • Use the "Delete All Data" button in the Privacy & Data tab to clear all records.
    • Use "Auto-purge old data" to automatically remove records older than a specified number of days.
    • Search and delete data for specific domains.

    No Account Required

    TimeDash does not have accounts. There is no login, registration, or cloud sync.
