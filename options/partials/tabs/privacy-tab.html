    Privacy Policy

    Last updated: April 2026
    Read Full Policy

    Overview

    TimeDash runs entirely in your browser. It does not include analytics, telemetry, ads, or any other form of remote tracking. There is no login, no account, and no server that receives your browsing history. All data processing happens locally on your device using your browser's built-in storage APIs.

    Data Storage

    TimeDash stores data in your browser profile using Chrome's chrome.storage.local API. This storage is scoped to the extension and your browser profile. No data is written outside of this API. The stored data includes:

    • Time tracking records and per-domain usage history
    • Playback speed preferences per domain
    • Site blocking and restriction rules
    • Theme, accent color, and other UI customization settings

    TimeDash reads this data solely to display your usage statistics and enforce your configured rules. No portion of this data is transmitted to any remote service.

    Network Activity

    TimeDash does not make any network requests of its own. If you see requests in DevTools, they originate from the browser's built-in favicon resolution or from extension-scoped resources. The extension itself contains no networking code that sends data to an external server.

    Permissions

    These are the permissions that TimeDash needs in your browser:

    1. storage: for settings and usage records only.
    2. tabs: for reading the active tab URL. Attributes time to the correct site and to apply block or restriction rules.
    3. scripting: injects video speed controls into HTML5 video players.
    4. webNavigation: tracks URL changes across frames.
    5. host_permissions: allows content scripts to run on the sites you visit. Without this, tracking and video controls would only work on a limited set of pre-approved domains.
    6. alarms: for scheduled maintenance (daily usage resets and optional auto-purge sweeps).
    7. notifications: (optional) alerts (e.g., daily time limit reached). Never used for marketing or promotional purposes.

    Data Deletion

    You retain full control over your data at all times. You can delete individual domain records or wipe everything from the Data Management section above. Uninstalling the extension automatically removes all stored data from your browser profile. No residual data remains on any remote system because no remote system has ever received it.

    • Use Delete All Data in the Data Management section above to clear every stored record immediately.
    • Enable Auto-purge Old Data to automatically remove records older than a configurable number of days.
    • Use the search field under Delete Specific Data to target records for a single domain.

    No Account Required

    TimeDash does not have accounts, logins, registration forms, or cloud sync. All data is stored exclusively on your device. You are solely responsible for it, and no one else can access it through the extension.

    Third-Party Code

    TimeDash uses a small number of open-source third-party libraries for development tooling (linters, formatters). These are not present in the distributed extension and cannot access your data. The extension runtime contains no third-party analytics, telemetry, or tracking libraries.
