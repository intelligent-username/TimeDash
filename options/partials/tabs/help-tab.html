Features Overview

Time Tracking

Keeps a running total of how long you spend on each site while you browse. Check the popup for today's numbers, or open Analytics for longer range trends and the heatmap.

Video Speed Control

Lets you change playback speed on pages that use HTML5 video players, including YouTube and Vimeo. You can also set shortcuts so you do not have to hunt for controls every time.

Site Blocking

Block sites completely, or set a daily limit. If you use restricted access rules, you can allow short sessions without removing the block.

Analytics Dashboard

Charts show your time by day, week, month, and year. The activity heatmap is a quick way to spot patterns without digging through pages of numbers.

Customization

Pick light, dark, or auto theme, choose an accent color, and set shortcuts that match how you actually use the browser.

Privacy First

Your data stays on your device. No accounts, no cloud sync, no tracking. Read the full policy

Data Export & Import

Export your data as JSON for backup. Import it later to restore settings or move to another browser profile.

Keyboard Shortcuts

Set speed control shortcuts in Video Control settings. Defaults: + increase, - decrease, . reset.

Getting Help

Report Issues or Suggest Features

Bug reports and feature requests go here: github.com/intelligent-username/TimeDash/issues.

Please include the site URL, the exact steps to reproduce, what you expected, what happened, and your browser version. Screenshots are useful when the UI looks wrong.

For video problems, say if the player is embedded and if it lives inside an iframe. If there is a console error, paste it.

View Source Code

Source code is at github.com/intelligent-username/TimeDash.

If you want to audit what TimeDash can access, start with manifest.json, then look at the background code and content scripts.

Privacy Questions

Start with the full Privacy Policy. If something does not match what you are seeing in the browser, open an issue and include details. A Network tab screenshot helps if you think a request is being made.
